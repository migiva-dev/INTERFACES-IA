# INTERFACES-IA

Repositorio con diferentes proyectos prácticos relacionados con el uso de Inteligencia Artificial aplicada a interfaces web, asistentes conversacionales, sistemas de pregunta-respuesta, corrección de textos, entrenamiento con datos propios y recuperación de información mediante bases de datos vectoriales.

Este repositorio está pensado como una recopilación de ejercicios y demos para aprender cómo integrar IA en aplicaciones web usando tecnologías como HTML, CSS, JavaScript, PHP y Python.

---

## 📌 Descripción del proyecto

**INTERFACES-IA** reúne varios proyectos independientes donde se trabaja con distintos conceptos de Inteligencia Artificial desde un enfoque práctico.

A lo largo del repositorio se pueden encontrar ejemplos de:

- Introducción al uso de IA.
- Creación de un mini chat tipo ChatGPT.
- Sistemas de pregunta y respuesta.
- Corrección automática de correos.
- Entrenamiento con datos en formato JSON.
- Uso de RAG con bases de datos vectoriales.
- Comunicación entre PHP y Python para integrar IA en una aplicación web.
