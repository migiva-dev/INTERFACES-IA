Proyecto 1

Introducción a IA: instalación de Ollama y modelos

En este primer proyecto se realiza una introducción al uso de inteligencia artificial ejecutada de forma local mediante Ollama. El objetivo es instalar la herramienta, descargar un modelo y comprobar su funcionamiento desde la terminal.

Objetivo del proyecto

El objetivo principal es preparar un entorno local para trabajar con modelos de inteligencia artificial sin depender directamente de una plataforma externa.

Para ello se utiliza Ollama, una herramienta que permite descargar y ejecutar modelos de lenguaje desde la terminal de Ubuntu.

Resumen: este proyecto sirve como base para los siguientes ejercicios, donde se conectará Ollama con PHP, formularios, JSON, RAG y comunicación entre lenguajes.

Tecnologías usadas

  • Windows 11
  • Ubuntu
  • Apache
  • Terminal de Linux
  • curl
  • zstd
  • Ollama
  • Modelo ministral-3:3b

Pasos realizados

1. Instalación de curl

Primero se instala curl, una herramienta que permite descargar contenido y realizar peticiones desde la terminal. 

sudo apt install curl

2. Instalación de Ollama

Después se instala Ollama mediante el script de instalación. 

curl -fsSL https://ollama.com/install.sh | sh

3. Instalación de zstd

En algunas instalaciones puede aparecer un error indicando que falta zstd. Esta herramienta permite descomprimir correctamente el paquete de Ollama. 

sudo apt update
sudo apt install zstd -y

4. Descarga del modelo

Una vez instalado Ollama, se descarga el modelo indicado para la práctica. 

ollama pull ministral-3:3b

5. Comprobación de modelos instalados

Este comando permite ver los modelos que hay descargados en el equipo. 

ollama list

6. Ejecución del modelo

Para comprobar que el modelo funciona, se realiza una pregunta directamente desde la terminal. 

ollama run ministral-3:3b "Quien es Jose Vicente Carratalá?"
Resultado esperado: si la instalación es correcta, Ollama responderá desde la terminal utilizando el modelo descargado.

Comandos principales del proyecto

  • sudo apt install curl
  • curl -fsSL https://ollama.com/install.sh | sh
  • sudo apt update && sudo apt install zstd -y
  • ollama pull ministral-3:3b
  • ollama list
  • ollama run ministral-3:3b "Quien es Jose Vicente Carratalá?"

Conclusión

En este primer proyecto se ha preparado el entorno necesario para trabajar con inteligencia artificial en local. Se ha instalado Ollama, se ha descargado un modelo y se ha comprobado que puede responder preguntas desde la terminal.

Este proyecto es importante porque será la base de los siguientes ejercicios, donde se usará Ollama desde aplicaciones web realizadas con PHP y otros lenguajes.
